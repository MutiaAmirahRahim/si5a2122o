  1. CARA INSTALL WordPress.org Buka alamat wordpress.org kemudian pilih tombol unduh wordoress.5.8.1

    Isi koneksi database

    Nama pengguna: diisi dengan root

    Nama situs : judul utama dari website

    Jika proses instalasi sudah berhasil, login ke dashboard WordPress
