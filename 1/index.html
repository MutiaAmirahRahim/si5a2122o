CARA INSTALL WordPress.org
1. Buka alamat wordpress.org kemudian pilih tombol unduh wordoress.5.8.1
2. Buka file kemudian ekstrak
3. Rename folder yang telah di ekstrak Contoh : si5a2122o (tanpaspasi)
4. Pindahkan folder xamps ke dalam folder htdocs
5. Buka browser kemudian ketikkan URL : http://localhost/nmfolderdihtdocs/wp-admin
6. Pilih bahasa tekan tombol Continue
7. Menyiapkan database, Klik ayo.
8. Mengisi Database dengan membuka xampp lalu klik di MySQL admin.
9. Buat database
10. Isi koneksi database
Name basis data : diisi dengan nama database
Nama pengguna: diisi dengan root
Sandi : dikosongkan. Kemudian klik tombol Submit.
11. Masukkan detail mengenai wordpress,
Nama situs : judul utama dari website
Nama pengguna : username yang akan digunakan untuk login ke dashboard WordPress
Sandi : password yang akan digunakan untuk login ke dashboard WordPress. Kemudian klik install wordpress
12. Jika proses instalasi sudah berhasil, login ke dashboard WordPress
dengan cara klik tombol Login dan masukkan username dan password yang dibuat sebelumnya.

CARA BACKUP
1. Buat satu folder baru
2. Copy file dalam folder htdocs (si5a2122o), pindahkan ke dalam folder yang kita buat tadi.
3. Kemudian dari phpMyadmin klik exsport pilih costum dan nama database yg telah dibuat tadi
(db_si5a2122o) kemudian klik go
4. Masukkan file- file yang telah didowload ke dalam folder yang telah dibuat dan compress jadi satu

RESTORE WordPress
1. Extract file yang telah di backup kemudian masukkan folder yang telah diextract ke dalam htdocs
2. Import file sql, kemudian pilih file, dan pilih set format (SQL)
3. klik go
4. Jika sudah berwarna hijau file SQL telah ada dan berhasil di restore
