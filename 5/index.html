Pemilihan Tema di Wordpress Pemilihan tema dikategorikan menjadi beberapa kolom. Untuk Menginstal Tema terbagi menjadi beberapa cara yaitu : Secara Online : Ke admin area Klik tampilan Pilih tema Kemudian klik tambah baru Lalu pilih tema yang akan diterapkan Klik install Untuk mengaktifkan tema klik tombol aktifkan dan import Secara semi online : Klik pratinjau (Bard oleh Wp Royal) Cari Copy judul dari tema tersebut di google contoh pencarian https://wordpress.org/themes/bard/ Klik install Kemudian download Untuk menginstall tema ini : Dengan Meng-upload file zip yang sudah di download sebelumnya. Klik tampilan Cari tema Klik tambah baru Klik unggah tema Klik install Dan aktifkan Pindahkan folder tema : File yang telah didownload diekstrak Kemudian masukkan folder ke dalam htdocs->wp-content->themes Costum Menu Bisa dibuat dari laman, pos, kategori dan link.
